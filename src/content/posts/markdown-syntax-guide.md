本指南将通过制作美味意面的过程，向您展示如何使用Markdown格式化文本。让我们在美食世界中探索Markdown语法！

## 基础语法演示

### 标题层级

食谱中常用多级标题组织内容：

```markdown
# 经典蒜香培根意面
## 食材准备
### 主要食材
#### 调味料
##### 厨房工具
###### 小贴士
```

### 文本强调

描述食谱时需要突出关键信息：

**重要步骤** 使用粗体  
*关键时间点* 使用斜体  
***特别注意事项*** 使用粗斜体  
~~可选步骤~~ 使用删除线

### 列表应用

#### 食材清单（无序列表）：

- 意大利面 400克
- 培根 200克
  - 建议肥瘦相间
  - 切成小片
- 蒜末 3-4瓣
- 橄榄油 2汤匙

#### 烹饪步骤（有序列表）：

1. 烧开盐水
2. 放入意面煮至8分熟
3. 同时准备酱汁：
   1. 热锅倒油
   2. 爆香蒜末
   3. 加入培根翻炒

### 引用块

> 主厨秘诀：煮面水加少许橄榄油可防粘黏  
> 保留适量面汤可用于调节酱汁浓稠度

### 代码块

精确展示配料比例：

```python
# 食材计算器
def calculate(人数):
    基准面量 = 100  # 克/人
    基准培根 = 50   # 克/人
    
    return {
        "意面": 基准面量 * 人数,
        "培根": 基准培根 * 人数
    }
```

热量计算公式：

$$
总热量 = \sum(食材重量 \times 每克热量)
